# Metis Voron 2.4R2 Klipper Backup

![image](./assets/metis.jpg)

## Physical setup
 ### Hotend
- Mellow3D FLY-SHT36 v2 [Follow Maz's fantastic guide here](https://maz0r.github.io/klipper_canbus/)
- E3D Revo Voron
- A4T toolhead with WWBMG (dual sensor)
- Beacon H

### Motion
- LDO 0.9deg A/B & LDO 1.8deg Z
- LDO rails 

### Electronics
- Octopus Max EZ
- MKS Canable Pro with [Reapola's awesome DIN mount](https://github.com/VoronDesign/VoronUsers/tree/master/printer_mods/reapola/MKS_CANable_Pro_mount)
- Raspberry Pi 5 4Gb 
- ESP32 powering WLED


### Physical Mods
- Common Anomaly X Beam
- TI Backers 
- GE5C bearings for Z 
- Vitali3D CNC idlers, AB mount & xol bracket
- Whoppingpochard's Maxwell Kinematic mounts 
- Purge Bucket & Bambulab silicon brush
- BTT TFT 5.0
- Chamber thermistor on chain guide
- Corsair LED lighting
